The SNAP API Source Code Examples

The examples are available in the snap-examples repository on GitHub. Provided here is a short documentation for the Java source code examples provided along with the SNAP Software. The examples show the basic concepts how to use the SNAP Java API for your own applications or how to extend SNAP by writing new modules.

Very simple examples showing how to read data from a product and write them out:

  • BandWriterMain.java
  • BitmaskWriterMain.java
  • TiePointGridWriterMain.java

A very simple NDVI (normalized differential vegetation index) processing implementation. The package contains a single source file:

  • NdviSimpleMain.java

A more advanced processor which uses parts of the SNAP Graph Processing Framework (GPF) can be found in the GitHub examples at snap-engine-java-operator. This example is desgned to be build as SNAP module to be separately added to an existing SNAP installation.

SNAP can also understand Python code. If you like to implement a processor in Python this example is for you: snap-engine-python-operator

Newcomers who want to develop export tools and/or data processors are invited to look at the Developer Guide which we have prepared in our wiki.

This is the list of the most important API packages used by the examples:

  1. The {@link org.esa.snap.core.datamodel} package.
  2. The {@link org.esa.snap.core.dataio} package.
  3. The {@link org.esa.snap.core.gpf} package.
  4. The {@link org.esa.snap.util} package.
